# YAGNI (You Aren't Gonna Need It)

**মূল কথা:** ভবিষ্যতে লাগতে পারে ভেবে আজই কোনো কোড লিখবেন না।

**বিস্তারিত ব্যাখ্যা:**
আমরা অনেক সময় ভাবি, "ভবিষ্যতে হয়তো এই ফিচারটি লাগতে পারে, তাই আজকেই এর জন্য কিছু এক্সট্রা ক্লাস বা মেথড লিখে রাখি।" কিন্তু বেশির ভাগ সময়ই সেই ফিচারগুলোর আর দরকার হয় না, ফলে কোডবেস অযথাই বড় এবং জটিল হয়ে যায়। YAGNI প্রিন্সিপাল বলে, বর্তমানে আপনার যে রিকোয়ারমেন্ট আছে, ঠিক ততটুকুই কোড লিখুন। 

**C# উদাহরণ:**
এই ফোল্ডারের `YAGNI.cs` ফাইলে বর্তমান চাহিদার ওপর ফোকাস করে কীভাবে কোড লিখতে হয়, তার C# কোড উদাহরণ দেওয়া আছে।
