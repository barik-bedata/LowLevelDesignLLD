# KISS (Keep It Simple, Stupid)

**মূল কথা:** কোড যতটা সম্ভব সহজ রাখুন।

**বিস্তারিত ব্যাখ্যা:**
সফটওয়্যার ডেভেলপমেন্টে সব সময় চেষ্টা করা উচিত যেন সিস্টেম ডিজাইন এবং কোড যতটা সম্ভব সহজ থাকে। অযথাই জটিল লজিক বা ওভার-ইঞ্জিনিয়ারিং করার কোনো মানে নেই। কোড এমন হওয়া উচিত যেন অন্য কোনো ডেভেলপার বা ভবিষ্যতে আপনি নিজে দেখলে খুব সহজেই বুঝতে পারেন যে কোডটি কী কাজ করছে।

**C# উদাহরণ:**
এই ফোল্ডারের `KISS.cs` ফাইলে জটিল কোডকে কীভাবে সহজভাবে লেখা যায়, তার C# কোড উদাহরণ দেওয়া আছে।
